jmeter性能测试简介
该测试使用jmeter3.0版本，并使用jmeter-plugins插件辅助图像、数据统计对**系统进行压力/性能测试。
1.为避免录制脚本出现冗余代码较多的情况，该测试使用手工书写脚本；
2.为验证返回数据等正确性，该测试使用正则、断言等进行验证返回值/结果；
3.该测试结果数据包括两部分：a.服务器指标(CPU/内存/带宽等重要指标)，b.业务指标(吞吐量/响应时长/错误率等重要指标)
